ppangteul

영상 제작 워크플로우를 자동화하는 웹 기반 도구

주요 기능

  • 자막 추출 — 영상에서 자막을 자동으로 추출
  • 스크립트 작성 — AI를 활용한 영상 스크립트 자동 생성
  • 이미지 생성 — 스크립트에 맞는 이미지를 AI로 생성
  • 모션 적용 — 이미지에 자동 모션 효과 적용
  • TTS 생성 — 스크립트를 음성으로 변환
  • 영상 합성 — 이미지, 모션, TTS를 조합하여 최종 영상 생성
  • 썸네일 제작 — AI 기반 썸네일 디자인 및 제안
  • 영상 업로드 — 완성된 영상을 채널에 직접 업로드
  • 채널 분석 — 조회수, 시청 시간, 구독자 등 분석 데이터 제공

기술 스택

Backend
Python, FastAPI
Frontend
HTML, JavaScript
AI/ML
Google Gemini API, WhisperX
영상 처리
OpenCV, FFmpeg

API 사용

이 애플리케이션은 YouTube Data API v3와 YouTube Analytics API v2를 사용합니다. 모든 작업은 사용자의 명시적 동의(OAuth 2.0 인증)를 통해 수행되며, 사용자 본인의 채널에 대해서만 작동합니다.
